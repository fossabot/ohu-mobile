# Form: Input


## Usage

```js
import { Input } from '@ohu-mobile/core'
```


## Design Goals

1. ✅Support adding icon or text in `left` and `right`.

2. ✅Customized `Input` type: `password`.

3. ✅Different style in `Form.Field`.

4. ✅Align: `left`, `center`, `right`.

5. ✅Support `disabled`, `readonly` etc.
