## Install & Usage

### Install Components

```bash
npm install @ohu-mobile/core

# or

yarn add @ohu-mobile/core
```

### Install SVG Icons

```bash
npm install @ohu-mobile/icons

# or

yarn add @ohu-mobile/icons
```


### Install Sass Mixins Library

```bash
npm install @ohu-mobile/styles

# or

yarn add @ohu-mobile/styles
```
